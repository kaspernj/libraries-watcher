# Development Notes

## Commands
- `npm run lint`: Not configured in this project (no `lint` script in `package.json`).
- `npm run typecheck`: Runs TypeScript without emitting files.
- `npm run test`: Runs the Jasmine test suite.

## Tips
- Use `npm run build` to produce the `build/` output if you need compiled artifacts.
- Use `gh pr create` to open a PR from the current branch.
- Add or update tests for new/changed functionality.
- Run `npm run lint` and `npm run typecheck` after making changes.
